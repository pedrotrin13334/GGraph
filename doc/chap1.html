1 The Graph defined on Groups Package

The GGraph is a GAP; package dedicated to implement methods, functions and constructions for different types of Graphs defined on Groups.

To utilize the GGraph package, it's necessary to download it from the main GGraph repository in github in the .gap/pkg directory, then type in the console:

gap> LoadPackage("GGraph");
